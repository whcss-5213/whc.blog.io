全局变量 ​

  • global

    类似window对象

  • __filename

    当前正在执行脚本的文件名

  • __dirname

    当前正在执行脚本的目录路径

js
console.log(global);
console.log(__filename); // d:\Desktop\node\index.js
console.log(__dirname); // d:\Desktop\node

path模块 ​

  • path.basename('d:/a/b/c.html') c.html

  • path.dirname('d:/a/b/c.html') d:/a/b

  • path.extname('d:/a/b/c.html') .html

  • path.join()

    js
    path.join(...'/abc') // \a\b\c
path.join(...'abc') // a\b\c
