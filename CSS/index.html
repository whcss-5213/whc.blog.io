-child : 按照子元素的顺序进行筛选,

-of-type :　先按照子元素的类型分类，在按照同类型标签的顺序进行筛选

css
/* 偶数 */
nth-child(even)
nth-of-type(even)
/* 奇数 */
nth-child(odd)
nth-of-type(odd)
/* 从第五个到最后 */
nth-child(n+5)
nth-of-type(n+5)
/* 前六个 */
nth-child(n-6)
nth-of-type(n-6)

p:only-of-type	/*选择所有仅有一个子元素为p的元素*/
p:only-child	/*选择所有仅有一个子元素为p的元素*/

11.伪类选择器 ​

:checked 表单被选中

:focus 表单获取焦点

:focus-within 当元素或其任意后代元素被聚焦时

:link 未访问

:active 被激活

:visited 被访问后

:hover 鼠标悬浮

:target 突出显示活动的 HTML

12伪元素选择器 ​

  2. 是行内元素，一般需要加 display:block
  3. 通常需要定位
css
