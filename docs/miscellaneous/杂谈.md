# 杂谈

## 时间

我发现一个超治愈的「沉浸式消磨时间」妙招

选一趟运营时长超一小时的长线公交，

始发站花1.4元落座，挑靠窗能充电的绝佳位置，

慢悠悠一路坐到终点站，再凭免费换乘返程。

三小时轻松打发完毕～

既不是宅家躺平式虚度，

又自带忙碌氛围感，悄悄放空身心，惬意又自在

## 梦想

多年以后再谈及梦想，我总会想起初中语文课的那个上午。

当时老师布置了三分钟梦想演讲，可我压根没什么明确梦想。没法直白敷衍，只好说人的梦想会随着人生阶段的变化而变化的，分阶段细数小时候、上学后的各种小期许，勉强凑够了时长。

岁月辗转至今，回头细看，我依旧活成了没有执念理想的模样，只愿随遇而安、随日子浮沉。我时常疑惑缘由，难道是自己心态出了症结？

活着已经不容易了，还谈什么梦想呢？
