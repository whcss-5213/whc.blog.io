给你整理最常用、质量最高、可商用的 Lottie 动画资源网站，按优先级排序👇
 
🌟 1. LottieFiles（官方首选，必收藏）
 
- 网址：https://lottiefiles.com/ 
- 资源：66万+ 免费+付费动画，覆盖UI、加载、图标、节日、业务场景 
- 优势：
- 在线预览、一键改色、编辑动画

- 免费大多可商用（Simple License，无需署名）
- 支持 Figma 插件、AE 导出、在线编辑
- 适合：Web/小程序/App 全场景

- 网址：https://iconscout.com/lottie

- 优势：风格清新，适合启动页、空状态、引导页
 
🧩 4. Lottieicon（纯动态图标库）
